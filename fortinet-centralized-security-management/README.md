# Fortinet Centralized Security Monitoring & Management

Enterprise-focused centralized network security monitoring and management implementation using FortiManager and FortiAnalyzer.

The project focuses on centralized firewall management, ADOM administration, policy deployment, VPN and SD-WAN concepts, traffic analysis, log monitoring, and enterprise network security operations within a multi-site environment.

---

## Current Project Scope

- Centralized device management using FortiManager
- Centralized log monitoring using FortiAnalyzer
- ADOM configuration and device grouping
- Firewall policy deployment and validation
- IPsec VPN and Hub-and-Spoke architecture
- SD-WAN and SLA monitoring concepts
- Traffic analysis and security monitoring
- ADVPN branch-to-branch communication concepts
- Enterprise security policy management

---

## Technologies

- FortiManager
- FortiAnalyzer
- FortiGate
- SD-WAN
- IPsec VPN
- ADVPN
- VMware
