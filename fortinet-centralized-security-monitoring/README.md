# 🚧 Fortinet Centralized Security Monitoring

## 📌 Project Status

This project is currently under active development as part of an enterprise-focused internship and academic implementation using Fortinet technologies. Additional configurations, monitoring workflows, and documentation are being continuously added.

---

## 📖 Overview

This project focuses on implementing centralized network security monitoring and management using Fortinet technologies including FortiManager, FortiAnalyzer, and FortiGate devices.

The implementation follows a centralized security architecture designed to improve visibility, simplify administration, support policy management, and enhance threat monitoring across multiple network devices.

---

## 🛠 Technologies Used

- FortiManager
- FortiAnalyzer
- FortiGate
- FortiView
- VMware
- Network Security Policies
- Firewall Management
- Security Monitoring
- Log Analysis
