# Lessons Learned

## Overview

This project provided practical hands-on experience in deploying, configuring, and managing a centralized SIEM environment using Wazuh within an enterprise-style infrastructure.

The implementation improved technical understanding across endpoint monitoring, security event analysis, cloud audit integration, and operational security monitoring workflows.

---

# Technical Lessons Learned

## Centralized Security Monitoring

One of the most important lessons learned was the value of centralized visibility across endpoints, servers, and cloud services.

The implementation demonstrated how SIEM platforms can:
- Aggregate logs from multiple systems
- Simplify security monitoring
- Improve threat visibility
- Support faster investigation workflows

---

## Enterprise Agent Management

Deploying and managing agents across multiple systems highlighted the importance of:
- Standardized deployment methods
- Centralized configuration management
- Automated onboarding approaches
- Operational scalability

Using Group Policy Object (GPO) deployment improved understanding of enterprise-scale endpoint management practices.

---

## File Integrity Monitoring (FIM)

The File Integrity Monitoring implementation demonstrated how monitoring file changes can improve:
- Threat detection
- Visibility into suspicious modifications
- Incident investigation
- Security auditing capabilities

This also reinforced the importance of monitoring sensitive application and web directories.

---

## Office 365 Security Monitoring

Integrating Office 365 audit monitoring provided valuable exposure to:
- Cloud security monitoring
- Azure AD integrations
- API-based log collection
- Authentication event monitoring
- Hybrid security visibility

This improved understanding of how SIEM platforms can extend monitoring into cloud environments.

---

## Dashboard Engineering & Visibility

Designing custom dashboards improved understanding of:
- Security event visualization
- Alert prioritization
- SOC operational workflows
- Threat analysis efficiency

The dashboard implementation highlighted the importance of presenting security data in a structured and actionable manner.

---

# Operational Lessons Learned

## Security Documentation

The project reinforced the importance of:
- Clear technical documentation
- Standardized configurations
- Organized deployment procedures
- Proper security documentation practices

---

## Security & Privacy Considerations

Working within a real-world environment emphasized the importance of:
- Protecting sensitive organizational information
- Redacting operational data
- Securing credentials and infrastructure details
- Following responsible security practices

---

# Conclusion

This project significantly improved practical SIEM deployment knowledge, security monitoring capabilities, endpoint visibility understanding, and operational SOC-related skills through hands-on enterprise-focused implementation experience.
